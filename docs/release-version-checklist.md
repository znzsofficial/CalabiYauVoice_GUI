# Release Version Checklist

This project currently stores the app release version in a few separate places. When preparing a new release, update all entries below in the same change.

## Version Fields

- `androidApp/build.gradle.kts`
  - `defaultConfig.versionCode`: Android monotonically increasing integer, for example `24`.
  - `defaultConfig.versionName`: user-facing Android version, for example `"2.0.4"`.

- `desktopApp/build.gradle.kts`
  - `compose.desktop.application.nativeDistributions.packageVersion`: desktop package version, for example `"2.0.4"`.

- `androidApp/src/main/kotlin/com/nekolaska/calabiyau/feature/settings/AboutScreen.kt`
  - About page display text, for example `Version 2.0.4`.

- `androidApp/src/main/kotlin/com/nekolaska/calabiyau/feature/settings/SettingsScreen.kt`
  - Fallback version string used if Android package info cannot be read.

## Update Metadata

- `downloadPage/downloads/latest.json`
  - `versionName`
  - `versionCode`
  - `changelog`
  - `publishedAt`

- `downloadPage/dist/downloads/latest.json`
  - Keep this copy in sync with `downloadPage/downloads/latest.json` if the built download page is committed.

## What Not To Edit

- Do not update dependency versions that only happen to match the app version, such as entries in `package-lock.json`, `yarn.lock`, `node_modules`, `build`, or `.git` logs.
- Do not edit generated Android build outputs under `androidApp/build`; they are regenerated by Gradle.

## Quick Search

Before finishing a release bump, search tracked source files for the old app version and old Android version code. Useful patterns:

```text
2.0.3
versionCode = 23
"versionCode": 23
packageVersion = "2.0.3"
versionName = "2.0.3"
Version 2.0.3
```

Adjust the old values to match the release you are replacing.

## Verification

Run at least the Android Kotlin compile check after changing release metadata:

```powershell
.\gradlew.bat :androidApp:compileDebugKotlin
```

If desktop packaging metadata changed and you need stronger confidence, run the relevant desktop packaging task before publishing.
